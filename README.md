## THE DNAI EQUATION

```
D = T[ C(G_AI, H) * G_AI ]

subject to:
  E(D, H) >= E_min        [ethical-use floor]
  R_w(D, H) = 0           [zero workforce replacement -- hard constraint]
```

| Symbol | Full Name | Role | Biological Analog |
|--------|-----------|------|-------------------|
| G_AI | Genetic AI Capability | Raw capability space | Genome / DNA |
| H | Human Centrality | Oversight and override authority (0-1) | Conscious vs. autonomic control |
| C(G_AI, H) | Constraint Matrix | Architectural brakes (caps-as-physics, LOTO, red lines) | DHODH metabolic brake |
| T | Tool-Usage Operator | Forces tool mode; prohibits autonomous goal-setting | Tissue-specific gene expression |
| D | Deployed System | D = T[C(G_AI, H) * G_AI] | Phenotype |
| E(D, H) | Ethical-Use Score | Rights, safety, transparency, fairness | -- |
| R_w(D, H) | Replacement Potential | Must equal **zero** -- hard constraint | -- |
| Psi(H) | Psychological Integrity | Worker well-being; degrades when R_w > 0 | Organismal homeostasis |

**Psychiatric Corollary:** R_w(D, H) > 0 => Psi(H) decreases
*(Any non-zero replacement potential produces measurable psychological harm -- McNamara & Thornton, 2025)*

**Displacement Creep Theorem:** For any epsilon > 0, if R_w = epsilon is permitted, then under
cost-minimization pressure, R_w -> max as t -> infinity. R_w = 0 must be enforced architecturally.

**3. Unified Equation-Governance-Seaworthiness Bundle**

DNAI delivers all three as a single deployable instrument with empirical validation
(Mercy Ridge pilot, 2026).
